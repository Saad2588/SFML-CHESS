SFML CHESS by Saad Abdullah (Roll No 2588)


A simple attempt at a chess game using the C++ SFML library


Includes:

-Turn Based Movement
-Move validation
-Check detection

& many more!


Controls:

Click on piece and then click on valid move square to move. You may only move the piece in accordance with the move order.


Installation:

-Requires C++17 or later
-SFML[2.6]

Compilation:

Linux :

g++ main.cpp -o chess -lsfml-graphics -lsfml-window -lsfml-system

./chess


Windows :

g++ main.cpp -o chess.exe -lsfml-graphics -lsfml-window -lsfml-system
chess.exe


Note:
Make sure to have all the assets downloaded alongside!
